Neste trabalho foi criado um banco de dados com o intuíto de atender a uma clínica odontológica, fazendo melhorias no atendimento e na gestão dos clientes. Os alunos integrantes usaram o Sistema de Gerenciamento de Banco de Dados(SGBD): PostgreSQL. Ótima oportunidade para desenvolvermos nossas técnicas nesse módulo da Residência em TIC/Software do Serratec. Utilizamos:

- **<h4>Requisitos funcionais:</h4>** Criação de tabelas, entidades, atributos e suas devidas restrições, relacionamentos e cardinalidades. 
- **<h4>Requisitos não funcionais:</h4>** Modelos conceitual, lógico e físico, insert, update, index e delete.
- **<h4>Consultas SQL:</h4>** count, view, sum, join, avg, max, min. 
- **<h4>Repositório Github:</h4>** Utilização do Git e Github para melhor versionamento e modificações do projeto.
